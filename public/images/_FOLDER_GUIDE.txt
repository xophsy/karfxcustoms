KAR FX CUSTOMS — IMAGE FOLDER GUIDE
=====================================
Replace any image by dropping in a new photo with the EXACT SAME filename.
No code changes needed when swapping images.

NEW: SERVICE PAGE IMAGES
/services/{category}/{service}/
─────────────────────────────────────────────────────────────
hero.jpg             → Page hero / banner image
gallery-01.jpg       → First gallery image
gallery-02.jpg       → Second gallery image
gallery-03.jpg       → Third gallery image
detail-01.jpg        → Close-up / detail shot
